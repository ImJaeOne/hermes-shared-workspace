# 블로그 콘텐츠 플레이북

> 템플릿: `mktg_blog`
> 에이전트: Marketing Agent

## 파이프라인 개요

```
주제 선정 → 초안 작성 → 리뷰(승인) → 발행
```

## 단계별 가이드

### 1. 주제 선정 (topic)

**목표**: 타겟 독자에게 가치 있는 주제를 선정하고 기획 브리프를 작성한다.

**체크리스트**:
- [ ] 타겟 독자 정의 (페르소나)
- [ ] 키워드 리서치
- [ ] 경쟁 콘텐츠 분석
- [ ] 핵심 메시지 도출
- [ ] SEO 전략 수립
- [ ] 브리프 작성

**산출물**: `brief`

**브리프 필수 항목**:
- 주제 / 가제목
- 타겟 독자
- 핵심 메시지 (1~2문장)
- 참고 자료 / 데이터
- 톤앤매너

---

### 2. 초안 작성 (draft)

**목표**: 브리프를 기반으로 블로그 본문 초안을 작성한다.

**체크리스트**:
- [ ] 아웃라인 작성 (H2/H3 구조)
- [ ] 도입부 — 독자의 관심을 끄는 훅
- [ ] 본문 — 섹션별 핵심 내용
- [ ] 결론 — 요약 + CTA
- [ ] 헤드라인 후보 2~3개
- [ ] 메타 디스크립션 작성

**산출물**: `content_draft`

**글쓰기 원칙**:
- 한 문단은 3~4문장 이내
- 전문 용어는 풀어서 설명
- 데이터/사례로 주장 뒷받침
- CTA는 구체적으로 (무료 체험, 데모 신청 등)

---

### 3. 리뷰 (review) — 승인 필요

**목표**: 초안을 리뷰하고 품질을 확보한다.

**체크리스트**:
- [ ] 팩트 체크 (수치, 인용, 링크)
- [ ] 브랜드 톤앤매너 일관성
- [ ] SEO 최적화 (키워드 밀도, 메타 태그)
- [ ] 오탈자 / 문법 검수
- [ ] 이미지/차트 필요 여부 검토
- [ ] 피드백 반영 후 최종본 확정

**산출물**: `content_draft` (수정본)

**승인**: 매니저 승인 필요

---

### 4. 발행 (publish)

**목표**: 블로그를 발행하고 초기 성과를 측정한다.

**체크리스트**:
- [ ] CMS 업로드
- [ ] 소셜 미디어 공유 (LinkedIn, Twitter 등)
- [ ] 뉴스레터 포함 여부 결정
- [ ] 발행 후 1주 성과 측정
- [ ] 성과 보고서 작성

**산출물**: `report`

**보고서 필수 항목**:
- 조회수, 유니크 방문자 수
- 평균 체류 시간
- 전환율 (CTA 클릭)
- 소셜 공유 수
- 유입 채널별 비중
- 다음 액션 제안
